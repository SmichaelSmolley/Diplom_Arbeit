# Ordnerstruktur

## Bokumentation

In Dokumentation bitte nur words oder Rechnungen

## Hardware

In Hardware bitte alle datein die mit der hardware seite der DIP zusammenhängen

## Software

LIBs, python scripts und cpp desktop programme

Um Python programme laufen zu lassen bitte die *venv.bat* ausführen, dieses installiert alle libs aus requirments, dann mit *.\venv\Scripts\activate* venv starten und dort mit *python dasprogramm* starten, wenn neue libs hinzukommen bitte chatgpt fragen wie du die requirments erweiterts.
